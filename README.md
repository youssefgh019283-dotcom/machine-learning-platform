# 🤖 Machine Learning Platform Desktop App

A specialized desktop interface for Machine Learning workflows. This project wraps a data science dashboard into a standalone executable using **Electron.js**.

## 🚀 Features
* **Desktop Integration:** Runs as a native `.exe` for Windows.
* **ML Dashboard:** Centralized interface for `machine_learning_platform.html`.
* **Clean UI:** Designed for focused data analysis without browser distractions.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript.
* **Framework:** Electron.js (Node.js backend).
* **Data Science Focus:** Designed to visualize Python/Machine Learning outputs.
